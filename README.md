# 蘭畹清芬電子詩集製作計畫
這計畫預備幫聶懋戡先生已出版的兩本「蘭畹清芬」詩集進行電子化。

## 計畫目標
本計畫目前預計有三個目標，
1. 將現有於 1994 年和 1998 年出版的兩集「蘭畹清芬」簡體版詩集，進行掃描、轉換、與校對後，製作成 EPUB 格式的電子書。
2. 將這兩本簡體中文版的詩集，重行轉換回正體中文版詩集，並再與留存之手稿重行校對一次。
3. 蒐集、補錄聶懋戡先生生前手稿中尚未發表的作品，重新集結之後製作成新電子版詩集發行。
